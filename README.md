## Next.js Ecommerce Store

Unique Adventures is a full-stack fictional adventure shop.

## Technologies used

 - Next.js
 - TypeScript
 - Cookies
 - CSS-in-JS (Emotion) 
 - PostgreSQL 
 - Unit tests & E2E tests with Jest and Puppeteer
